## 캐릭터 컨셉:
- **탐정**: 돋보기를 들고 세심하게 관찰하는 모습
- **과학자**: 실험복을 입고 색상 샘플을 분석하는 모습  
- **마법사**: 마법 지팡이로 색상을 변환하는 마법적인 모습
- **분석가**: 차트나 그래프를 보며 분석하는 전문적인 모습
- **아티스트**: 팔레트와 붓을 들고 색상을 조합하는 창작적인 모습

## 스타일 가이드:
- 친근하고 귀여운 만화 스타일
- 히잡을 착용한 여성 캐릭터로 통일
- 밝고 따뜻한 색조 사용
- 각 캐릭터별 고유한 의상과 소품
- 표정은 집중하되 친근한 느낌

- PNG 또는 JPG
